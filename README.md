# AI-Based Building Inspection

This repository contains the full workflow of my PhD research on AI-based computer vision for automated building inspection and structural health monitoring (SHM), covering all stages from literature review to final thesis.

## Project Scope
- Structural defect detection, localization, and progression prediction
- Image-based and drone-based inspection
- Deep learning and computer vision models
- Automated and intelligent building health monitoring
